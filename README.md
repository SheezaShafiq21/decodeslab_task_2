# Café Amore ☕

Café Amore is an enhanced version of my static coffee shop website, developed using **HTML5 and CSS3** during my **Frontend Development Internship at Decode Labs**. This updated version focuses on improving the overall user experience by introducing responsive design, better layout adaptability, and refined styling across different screen sizes.

## Overview

The project represents a fictional coffee shop website with a multi-page structure that allows users to explore the café, view the menu, learn about its story, and access contact details. The updated version includes improvements in responsiveness and visual consistency to provide a smoother experience on desktop, tablet, and mobile devices.

## Improvements Added

* Fully responsive layout for different screen sizes
* Improved mobile and tablet compatibility
* Enhanced page structure and styling
* Better spacing and alignment adjustments
* Refined user interface and visual presentation
* Improved navigation experience
* Updated responsive menu and content layouts

## Features

* Modern coffee-inspired design
* Multi-page website structure
* Semantic HTML5 elements
* Clean and organized CSS
* Interactive navigation
* Coffee menu with product cards
* Contact page
* Hover effects for better interaction
* Consistent branding and color palette

## Technologies Used

* HTML5
* CSS3
* Responsive Web Design (Media Queries)
* Visual Studio Code
